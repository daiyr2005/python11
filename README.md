Backend-аутентификация и авторизация (FastAPI)
Описание

Backend-приложение на FastAPI, реализующее собственную систему аутентификации и авторизации пользователей без полного использования встроенных механизмов фреймворка.

Реализована система разграничения прав доступа к ресурсам на основе ролей и разрешений.

Технологии

Python

FastAPI/ И знаю работат в django

PostgreSQL

SQLAlchemy

JWT

Pydantic

Функционал
Пользователи

Регистрация (ФИО, email, пароль)

Login / Logout (JWT)

Обновление профиля

Мягкое удаление пользователя (is_active = False)

Авторизация

Кастомная RBAC-система (User / Role / Permission / Resource)

Проверка доступа к эндпоинтам через зависимости FastAPI

401 — пользователь не аутентифицирован

403 — доступ запрещен

Администрирование

API для управления ролями и правами (только admin)

Назначение ролей пользователям

Mock-ресурсы

Тестовые эндпоинты для демонстрации работы системы доступа

Возвращают данные или ошибки 401 / 403
